Multi Layer Perceptron

A neural network as seen above is a structure that mimicks the human brain structure connection input nodes to other nodes. This technique forces an activation function on our input data. Since the same function is applied to each type of data we need standardization for our data in order to preserve accuracy. It also uses a techinque known as gradient descent and back propagation in order to minimize our loss function and create optimized predictions.
